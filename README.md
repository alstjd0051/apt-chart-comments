# apt-chart-comments

서울 아파트 실거래가 대시보드([data-viz](https://data-viz-fawn.vercel.app))의 **댓글 저장소**입니다.

## 역할

- [Utterances](https://github.com/utterance/utterances)를 통해 대시보드 댓글 페이지에 작성된 댓글이 **GitHub Issues**로 저장됩니다.
- 주제별로 별도 이슈가 생성되며, 이슈 제목은 해당 섹션 URL과 동일합니다.
